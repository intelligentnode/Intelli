This project is licensed under Apache-2.0. The code can be used freely, including commercially.

This document is **not** meant to block use of names like **Vibe Agents**. It exists to keep attribution clear and avoid confusion about what is “official”.

## Marks

Project marks include:
- Intelli / IntelliNode
- Vibe Agents

## Attribution request (especially for “Vibe Agents”)

If you use the term **“Vibe Agents”** publicly (docs, blogs, papers, talks, products, repos), please include a short attribution such as:

> “Vibe Agents” originated in the Intelli / IntelliNode project.

And link to one of:
- https://github.com/intelligentnode/Intelli
- https://docs.intellinode.ai/docs/python/vibe-agents

## Avoid confusion

You may use the marks, including “Vibe Agents”, as long as you do not imply:
- endorsement by the IntelliNode project, or
- that your fork/product is the official Intelli release.

If you fork or redistribute, please clearly state it is a fork/derivative and link back to the original project.
